I built this portfolio using React, Next.js and TypeScript.

The purpose of this portfolio is to showcase the projects
and mini projects I've created, as well as the learnings
I've gained along the way.

I used `next-mdx-remote` so that I could edit posts
without needing to write a bunch of HTML. I also used
Paco Coursey's <MDXAppLink href="https://github.com/pacocoursey/next-themes" openInNewTab>
next-themes </MDXAppLink>
for setting and updating the site's light/dark themes.

The minimalistic style of the home and project pages is from
Paco Coursey's <MDXAppLink href="https://paco.me/" openInNewTab>
portfolio </MDXAppLink>. The component for
displaying featured projects on the home page is from Jordan Singer's
Diagram <MDXAppLink href="https://diagram.com/" openInNewTab>
landing page </MDXAppLink>.

The site is deployed with Vercel.
