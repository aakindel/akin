I built this project using React. Next.js and TypeScript.

This purpose of this portfolio is to showcase the projects
and mini projects I've created, as well as the learnings
I've gained along the way.

I used next-mdx-remote to make it easier to edit the
information on the mini project pages.

The minimalistic style of the home page is inspired by
Paco Coursey's [portfolio](https://paco.me/), while the component for
displaying featured projects is from Jordan Singer's
Diagram [landing page](https://diagram.com/).

The site is deployed with Vercel.
