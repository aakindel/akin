Built with React, Next.js and TypeScript and `prism-react-renderer`.

The CodeBlock component is an easy to use component for displaying
syntax highlighted code blocks.

The component's style as well as the `props` for the component
are inspired by CodeSandbox's <MDXAppLink href="https://github.com/codesandbox/sandpack" openInNewTab>
Sandpack React component</MDXAppLink>.

The site is deployed with Vercel.
